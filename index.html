Mi portfolio

HTML5

Última versión del lenguaje de marcado HTML, utilizado para estructurar y presentar el contenido de páginas web.

CSS3

Última versión del lenguaje de hojas de estilo en cascada, que define la presentación visual de documentos HTML.

JavaScript

Lenguaje de programación de alto nivel, interpretado, que se utiliza principalmente para agregar interacciones a páginas web.

Bootstrap

Framework de código abierto, una plantilla o conjunto de herramientas para el diseño web, que facilita la creación de sitios web responsivos y adaptables a diferentes dispositivos.

NodeJS

Entorno de ejecución de JavaScript de código abierto y multiplataforma que permite a los desarrolladores crear aplicaciones del lado del servidor y de red con JavaScript.

React y Vite

Herramientas distintas utilizadas en el desarrollo web, que juntas crean un proceso de desarrollo más rápido y eficiente.

Git

Sistema de control de versiones distribuido, de código abierto, que se utiliza para rastrear cambios en archivos y coordinar el trabajo en proyectos de software.

Github

Plataforma basada en la nube donde puedes almacenar, compartir y trabajar junto con otros usuarios para escribir código.

Shell

Programa que actúa como interfaz entre el usuario y el sistema operativo, permitiendo interactuar con el sistema a través de comandos de texto.

CLI

Interfaz de usuario basada en texto que permite a los usuarios interactuar con un sistema operativo o un programa mediante la introducción de comandos.

SQL

Lenguaje de programación estandarizado utilizado para gestionar y manipular bases de datos relacionales.

MySql

Sistema de gestión de bases de datos relacionales de código abierto, utilizado para almacenar y gestionar datos.

MongoDB

Sistema de gestión de base de datos (DBMS) no relacional, de código abierto, que utiliza un modelo de datos orientado a documentos.

Wordpress

Sistema de gestión de contenidos (CMS) de código abierto y gratuito que permite a los usuarios crear y gestionar sitios web, blogs y tiendas online de manera sencilla, sin necesidad de conocimientos técnicos avanzados.
